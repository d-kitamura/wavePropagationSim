# Wave Propagation Simulator / 音波伝搬シミュレータ

## 概要

**Wave Propagation Simulator** は，ブラウザ上で動作する2次元音波伝搬シミュレータです。
点音源を自由に配置し，音波の干渉，位相差，距離減衰，ビームステアリング，グレーティングローブなどを視覚的・直感的に確認できます。

このアプリは，音響工学，波動，干渉，アレイ信号処理などの教育・デモンストレーション用途を想定しています。
HTML，CSS，JavaScriptのみで構成されており，インストール不要でブラウザから利用できます。

## 公開ページ

以下のGitHub Pagesから利用できます。

https://d-kitamura.github.io/wavePropagationSim/

## 主な機能

* 2次元空間における音波伝搬のリアルタイム可視化
* 点音源の配置，選択，削除
* 複数音源による干渉パターンの表示
* 音源ごとの振幅，周波数，位相，遅延の設定
* 位相指定と時間遅延指定の切り替え
* 瞬時音圧，RMS音圧，SPLの表示
* カラーマップ表示

  * Thermal
  * Diverging
  * Viridis
* 色スケールの固定・自動切り替え
* 等高線・節線の表示
* 距離減衰モデルの切り替え

  * 減衰なし
  * 1 / r
  * 1 / sqrt(r)
* 音速および温度の設定
* グリッド吸着による音源配置
* 観測点における音の再生
* 日本語・英語表示の切り替え

## 使い方

1. 公開ページをブラウザで開きます。
2. 「点音源配置」モードで音源を配置します。
3. 音源を選択し，振幅，周波数，位相，遅延などを調整します。
4. 表示モードや減衰モデルを切り替えて，音場の変化を観察します。
5. 「音圧観測」モードで観測点を設定すると，その位置での音を確認できます。
6. プリセットを選ぶと，代表的な音源配置をすぐに試すことができます。

## ローカルでの実行方法

このリポジトリをクローンまたはダウンロードし，`index.html` をブラウザで開いてください。

```bash
git clone https://github.com/d-kitamura/wavePropagationSim.git
cd wavePropagationSim
```

その後，`index.html` をブラウザで開きます。

簡易HTTPサーバを使う場合は，例えば以下のように実行できます。

```bash
python -m http.server 8000
```

ブラウザで以下にアクセスします。

```text
http://localhost:8000
```

# Wave Propagation Simulator

## Overview

**Wave Propagation Simulator** is a browser-based two-dimensional acoustic wave propagation simulator.
It allows users to place point sound sources freely and intuitively observe wave propagation, interference, phase differences, distance attenuation, beam steering, grating lobes, and related acoustic phenomena.

This application is designed for education and demonstration in acoustics, wave physics, interference, and array signal processing.
It is built only with HTML, CSS, and JavaScript, and can be used directly in a web browser without installation.

## Live Demo

The simulator is available via GitHub Pages:

https://d-kitamura.github.io/wavePropagationSim/

## Features

* Real-time visualization of two-dimensional acoustic wave propagation
* Placement, selection, and deletion of point sound sources
* Visualization of interference patterns generated by multiple sources
* Source-wise control of amplitude, frequency, phase, and delay
* Switching between phase-based and time-delay-based control
* Display modes:

  * Instantaneous pressure
  * RMS pressure
  * SPL
* Color map options:

  * Thermal
  * Diverging
  * Viridis
* Fixed or automatic color scale
* Contour and nodal-line display
* Distance attenuation models:

  * None
  * 1 / r
  * 1 / sqrt(r)
* Sound speed and temperature settings
* Grid snapping for source placement
* Audio playback at observation points
* Japanese and English user interface

## Presets

The following typical source configurations are available as presets:

* Single center source
* Two sources in phase
* Two sources in opposite phase
* Quadrupole
* Vertical line array
* Horizontal line array
* Beam steering
* Grating lobes
* Circular array
* Random phase array
* Moving source

## How to Use

1. Open the live demo page in a web browser.
2. Use the source placement mode to place sound sources.
3. Select a source and adjust its amplitude, frequency, phase, or delay.
4. Change the display mode or attenuation model to observe how the sound field changes.
5. Use the observation mode to set observation points and listen to the sound at those positions.
6. Select a preset to quickly test typical source configurations.

## Running Locally

Clone or download this repository and open `index.html` in a web browser.

```bash
git clone https://github.com/d-kitamura/wavePropagationSim.git
cd wavePropagationSim
```

Then open `index.html` in your browser.

You can also run a simple local HTTP server, for example:

```bash
python -m http.server 8000
```

Then access:

```text
http://localhost:8000
```

## Intended Use

* Lectures and exercises in acoustics
* Demonstrations of array signal processing
* Explanation of phase differences, delay-and-sum concepts, and beamforming
* Visualization of the relationship between source placement and sound fields

## Note

This simulator is intended for educational visualization.
It is not designed for precise acoustic analysis, architectural acoustic simulation, or prediction of real-world sound fields.
